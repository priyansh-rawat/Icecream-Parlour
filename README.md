# Icecream-Parlour
A C++ program to implement the services of an Icecream parlour 

### REQUIREMENTS 
```
The code was written in Dev C++ platform. The code should run well in most C++ supporting IDE. 
```
### IMPORTANT POINTS TO HELP RUN THE CODE
```
1) The password for editing the database is set as "password".
2) Entries can be both added and deleted from the database. An example database has been provided. 
3) The code was tested in just 1 PC. In case there's a problem in the visual part of code, it can be simply orrected by adding or removing '\t' escape sequence.
4) The code no '100' has been specifically set for the feature "MAKE YOUR OWN ICECREAM". 
5) The program offers a 10% discount for names starting with particular letters on every execution of code. The letters are generated randomly.
6) The code offers 3 choices to the customers. All the choices have been made to look realisitc but are static.
7) Exception handling has not been used in the project. At some places input of the wrong type may create run time exceptions.
```
